## 📋 一、项目概述

**AI Coworker Team** 是一个基于 LangGraph 的多智能体协作系统，模拟真实的团队开发环境。

### 核心愿景
让 AI Agents 像真实团队一样协作：分配任务、开发、测试、审核，完成整个软件开发生命周期。

## 🚀 五、快速开始

### 5.1 本地运行

```bash
# 克隆项目
git clone https://github.com/xiashao/ai-coworker.git
cd ai-coworker

# 安装依赖
pip install -r requirements.txt

# 启动服务
python main.py

# 打开浏览器
# http://localhost:8000
```

### 5.2 配置模型

1. 点击右上角 **设置** 按钮
2. 选择模型供应商（OpenAI/智谱AI/Anthropic）
3. 输入 API Key
4. 可选：为每个角色单独配置模型
5. 点击 **保存全部配置**

### 5.3 使用示例

```
# 创建项目
点击 "新建项目"

# 与 AI 团队对话
"创建一个用户登录功能"

# 智能体自动创建任务
[任务] 创建: 用户登录 | 实现登录注册功能 | dev | medium

# 开发完成任务
[任务] 状态: 用户登录 | review

# QA 审核通过
[任务] 状态: 用户登录 | done
```
